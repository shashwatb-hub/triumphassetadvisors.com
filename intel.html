📋 One-Time Setup Instructions
  1. Open your Hyderabad_Market_Brand_Database_v5 in Google Sheets (upload the Excel file to Drive → Open as Sheets)
  2. Click Extensions → Apps Script
  3. Delete any existing code. Paste the entire contents of triumph_apps_script.gs (downloaded with this HTML)
  4. Save the project. Name it Triumph Market API
  5. Click Deploy → New Deployment → Web App
    Execute as: Me | Who has access: Anyone
  6. Copy the Web App URL → paste it above → click Connect
  7. Back in the Apps Script editor, run setupMonthlyTrigger() once manually to activate the 1st-of-month auto-refresh
🔄 How Updates Flow
ON EVERY OPEN
HTML fetches live data from Google Sheets. Always fresh without manual updates.
1ST OF EVERY MONTH
Apps Script re-compiles all market sheets into Master Brand List. Automatically triggered.
CLAUDE VISION PUSH
Approved detections from field video are posted directly to Google Sheets with lat/lng.
MANUAL ADD BRAND
Add Brand form writes to Google Sheets instantly. Visible on map immediately.
